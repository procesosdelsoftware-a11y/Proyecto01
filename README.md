# GitHub Info – Todo sobre GitHub en español 

Sitio web educativo y bonito que explica qué es GitHub, quiénes lo usan, por qué es tan popular y cómo empezar en menos de 5 minutos.

¡Hecho con puro HTML, CSS y JavaScript vainilla!.


## Secciones del sitio

- Inicio  
- ¿Qué es GitHub?  
- ¿Quiénes lo usan?  
- ¿Por qué es tan popular?  
- Cómo empezar en GitHub (guía de 5 minutos)

## Cómo usarlo / desplegarlo tú mismo

1. Haz fork de este repositorio  
2. Ve a **Settings → Pages**  
3. En "Branch" selecciona `main` y carpeta `/root`  
4. ¡Listo! En menos de 1 minuto tendrás tu sitio en vivo

## Ideal para

- Aprender HTML/CSS/JS desde cero  
- Tener tu primer proyecto profesional en GitHub

- Practicar Git y GitHub Pages  
- Enseñar a otros qué es GitHub
